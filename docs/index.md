## Notes

- Jayess aims for JavaScript-like syntax, but it is not the JavaScript runtime.
- The compiler produces native executables.
- `var` and `const` are supported. `let` is not supported.
- `print(...)` still works, but it is deprecated in favor of `console.log(...)`.
